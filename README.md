## Description : 
This project uses the **Europarl French-English parallel corpus**, a collection of aligned texts drawn from European Parliament debates. It is used here to test a few predictions on the anglicism rise.

## Data

The data is **not included** in this repository due to its size.

You must download it manually from the official Europarl website:

1. Go to: [https://www.statmt.org/europarl/archives.html#v5](https://www.statmt.org/europarl/archives.html#v5)
2. In the **“Parallel corpus”** section, download the file:
   - **Parallel corpus French-English** (~176 MB, 04/1996–10/2009)
3. Extract the "fr-en" folder which contains two folders "en", "fr".
